SVG (and optional PNG) files in this directory are generated by the
[render-md-mermaid](https://github.com/marketplace/actions/render-md-mermaid)
workflow when `README.md` changes, or locally with `scripts/render-readme-mermaid.sh`
(Docker required). Do not edit SVGs by hand.

The markup must match **render-md-mermaid v3**: a `![alt](docs/images/….svg)` line,
then a `<details>` block (with a ` ```mermaid ` fence), so the action can find each
diagram. Edit the Mermaid source inside those `<details>` sections in `README.md`.
